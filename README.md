#  AI Money Mentor

##  Overview
AI Money Mentor is a simple Java-based application that helps users analyze their financial health. It calculates a financial score and provides smart suggestions based on income, expenses, savings, and debt.

---

## 🚀 Features
- Financial Health Score (0–100)
- Smart Suggestions
- User-Friendly Interface (Java Swing)
- Fast and Simple Analysis

---

##   Technologies Used
- Java
- Swing (GUI)

---

##   How It Works
1. Enter Income, Expenses, Savings, and Debt
2. Click "Analyze"
3. View your financial score
4. Get suggestions to improve your financial health

---

##   How to Run

### Prerequisites
- Install Java JDK 17
- Install VS Code or any Java IDE

### Steps
1. Clone the repository.
 2.   Open the project in your IDE

3. Run the file.


##  Example Output
- Score: 80/100  
- Suggestions:
- Reduce unnecessary expenses
- Increase savings
- Start investments
